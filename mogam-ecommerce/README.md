# Mogam E-commerce 🚀

Una aplicación de comercio electrónico moderna desarrollada con **React** y potenciada con **Firebase**, que cuenta con un catálogo dinámico sincronizado en tiempo real, autenticación de usuarios y un panel de administración protegido.

## 📋 Características Principales

*   **Autenticación de Usuarios:** Formulario de inicio de sesión y registro gestionado mediante Firebase Authentication.
*   **Rutas Protegidas:** El panel de administración está restringido y solo permite el acceso a usuarios autenticados. La persistencia está configurada para expirar al cerrar la pestaña/navegador.
*   **Base de Datos en Tiempo Real:** Catálogo de productos y carga de nuevos artículos completamente conectados a Firebase Firestore.
*   **Diseño Responsivo:** Interfaz adaptada para dispositivos móviles y computadoras utilizando Flexbox y Media Queries.
*   **Experiencia de Usuario:** Indicador de carga (Spinner) integrado mientras se obtienen los datos de la API de Firebase.

## 🛠️ Tecnologías Utilizadas

*   **React** (Vite)
*   **React Router DOM** (Manejo de rutas)
*   **Firebase** (Authentication & Firestore)
*   **CSS3** (Flexbox & Media Queries)
