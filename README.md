# Mogam E-commerce 🛍️

¡Bienvenido a **Mogam E-commerce**! Esta es una aplicación web moderna de comercio electrónico desarrollada con **React** y potenciada por **Vite**. Cuenta con una tienda interactiva para los usuarios y un panel de administración protegido para la gestión de productos, totalmente integrada con **Firebase** para la base de datos y autenticación.

---

## 🚀 Características Principales

*   **Tienda Interactiva:** Navegación por categorías de productos y vista de detalles en tiempo real.
*   **Carrito de Compras Global:** Gestión fluida de productos con cálculo de totales de forma centralizada.
*   **Panel de Administración (`/admin`):** Sección exclusiva y protegida para gestionar el catálogo de productos.
*   **Autenticación Segura:** Sistema de Login integrado con Firebase para proteger las rutas privadas.
*   **Diseño Adaptable:** Interfaz limpia y estilizada mediante CSS modular.

---

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** React (Hooks, Context API), Vite
*   **Enrutado:** React Router Dom
*   **Backend & Base de Datos:** Firebase (Firestore & Authentication)
*   **Estilos:** CSS3 nativo
*   **Despliegue:** Netlify
